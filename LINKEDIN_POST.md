Built a Price Watch Bot with Node.js and a live dashboard.

What it does:
- Tracks stocks and crypto with threshold rules
- Sends real-time Telegram alerts
- Includes a local web dashboard with manual "Check Now"

Tech:
- Node.js
- CoinGecko API (crypto)
- Stooq API (stocks)
- Telegram Bot API
